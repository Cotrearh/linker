Требуемые пакеты для работы версии python3 + pyqt4

Для Астра Линукс 1.4

Поставляются с ОС(в том числе со всеми зависимостями)
python3                 (протестировано на версиях 3.2.3-6)
python3-pyqt4           (протестировано на версии 4.11+dfsg-2astra1)
python3-pyqt4.qtsql     (протестировано на версии 4.11+dfsg-2astra1)
libqt4-sql-sqlite       (протестировано на версии 4:4.8.6+dfsg-1ubuntu2astra1) - для SQLite
libqt4-sql-psql         (протестировано на версии 4:4.8.6+dfsg-1ubuntu2astra1) - для PostgresQL
libc6, libc6-dev v2.13

Не поставляются с ОС

=========================================================

Требуемые инструменты для разработки

qtcreator - правка графической оболочки
IDEA\Sublime - правка кода
pyinstaller - сборка исполняемого файла

=========================================================

Конвертация файла графики, созданного в qtcreator, в код на python

pyuic4 spell.ui > spell_ui.py

=========================================================

Установка pyinstaller

sudo pip install pyinstaller

Примечание 2 - для корректной работы на определенной системе, исполняемый файл должен быть скомпилирован на целевой системе
(или на системе с аналогичным набором пакетов). Основная проблема в пакете libc6. Версии не имеют обратной совместимости.
Так при сборке с версией пакета 2.26 программа не запустится на пакете 2.13. Однако обратная ситуация допустима.
Например для работы на Астре нужно установаить проект на Астру и собрать его pyinstaller-ом.

=========================================================

Сборка проекта в один исполняемый файл под Линукс

pyinstaller --onefile --windowed  main.py
